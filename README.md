**本仓库所有内容中的代码均在`Mac`上运行的，可能部分和`Windows`有出入**
项目来源于[xuanhun师傅的PythonHackingBook1](https://github.com/xuanhun/PythonHackingBook1)，在他的基础上进行了一些个人理解然后对原文进行替换、对内容进行了一定的修改，添加了作业和实现代码，涉及到的代码也进行了一定的改进，后续将继续扩展，以帮助大家在各领域都能尽快的使用和理解`Python`，目前的大纲如下，各位也可以在`Issues`中进行补充或提出想法

# 1.基础部分
学习`Python`的基本的数值类型、数据结构以及语法等知识

# 2.进阶使用
在第一部分主要是介绍`Python`的语法、数据结构以及`OOP`概念，接下来则是对其进行实际的使用，用于安全开发等

# 3.安全领域
`Python`在网络安全领域应用广泛，常见的有`PoC`编写、爬虫以及木马编写等，在这里将会列举常用的部分

# 4.人工智能
`Python`在人工智能领域的应用，学习如何使用机器学习/深度学习/神经网络，并且编写代码进行模型训练

# 5.数据处理
`Python`在输出处理方面的应用，比如`Numpy`、`Pandas`以及自己针对安全领域的数据进行处理
